# Security Policy

## Source-Only Policy

This repository publishes source code only.

Do not attach, commit or distribute compiled `.ex5` binaries through this repository.

## Sensitive Files

Never commit:

- real `.set` presets;
- account numbers;
- broker exports;
- raw Strategy Tester reports;
- logs;
- optimization grids;
- `.env` files;
- credentials;
- local machine paths;
- screenshots containing account or broker data.

## Reporting Issues

If you find a security or privacy issue in the public repository, open a private communication channel with the maintainer before posting details publicly.

## Research Separation

Private research, real presets, live trading data, broker data and detailed optimization outputs must stay outside the public repository.
